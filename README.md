# Project Description

This project is a **full-stack web application** for managing real estate properties.  
It includes a **Frontend** for user interaction and a **Backend** with a REST API for handling data.

Users can:

* View and filter real estate listings  
* Add, edit, and delete properties (if authorized)  
* Leave reviews for properties  
* Navigate a modern and responsive interface

The backend is implemented using **FastAPI**, and the frontend is built with **React** and **Material UI**.

# Features

* **Property Management** – Create, read, update, delete properties  
* **Reviews** – Users can leave ratings and comments for properties  
* **Filtering & Sorting** – Filter by price, location, type; sort by price or date  
* **Pagination** – Load data in pages for faster performance  
* **Responsive UI** – Works on desktop and mobile devices  
* **API Documentation** – Automatically generated by FastAPI (Swagger / ReDoc)

# Running the Project

### 1. Clone the repository
git clone https://github.com/Zlatomyra/Website.git
cd real-estate-project

### 2. Backend Setup
cd backend
python -m venv .venv
.\.venv\Scripts\activate       # Windows
pip install -r requirements.txt
uvicorn main:app --reload

Server will run at:
http://127.0.0.1:8000

### 3. Frontend Setup
cd ../Frontend
npm install
npm start

Frontend will run at:
http://localhost:3000
